JS学习第三天

发表于 2021-09-15

第三天学习了条件语句，循环语句，对象的基本概念等等。

day3

流程控制语句

流程控制语句：代码按照什么顺序执行的语句。
默认语句：默认从上到下执行。
条件语句：多条分支中选择if if-else if-else-if switch
循环语句：for while do-while for-in
其他语句：break continue

分支语句

if

判断条件是否执行

if-else

if(条件){
条件为真的时候执行的代码;

}else{
条件为假时执行的代码;
}

if-else-if

switch

适用于条件和结果都单一时。
条件：判断某一个变量或者是某一个具体的值。
结果：指向的是一个具体的值。
break：防止穿透，如果不加，当条件匹配到一个符合的结果后，后于的结果都不匹配时，也会执行后边的代码。

分支嵌套

两个及以上分支语句嵌套使用

循环语句

将一段代码重复多次执行。

for

for(初始化变量; 循环结束条件; 变量更新){
执行的代码
}
初始化变量：var 变量=起始值; 数字0 1循环的次数从几开始的
循环结束的条件：条件为真时 循环执行 条件为假时 循环结束
变量更新：变量++ 变量–
注意：当心陷入死循环

变异for

初始化变量;
for(; 循环条件; ){
​ 循环体;
​ 变量更新;
}

注意：

  1. 变异格式中的两个分号不能省略。
  2. 初始化的变量都是全局变量。
  3. 循环执行完成后，变量会留在循环结束后的数值。

渲染页面

数据一般是通过数据请求的方式由后台返回给我们，页面的数据一般都不是固定的数据。
多个数据的时候，一般使用数组。
数组有长度有下标，通过下标的方式从数组中取出对应下标的值。
取值：数组[下标]

while

初始化变量;
while(条件){
​ 循环体;
​ 变量更新;
}

do-while

初始化变量;
do{
​ 循环体;
​ 变量更新;
}whlie(循环条件);

for和while

for明确循环次数，知道长度。
while不明确循环次数。

while和do-while

while先判断后执行。
do-while先执行后判断。

其他语句

break防止穿透，结束整个循环；
continue结束本次循环，开始下一次循环。

循环嵌套

for(外层初始化变量; 外层判断条件; 外层变量更新){
​ for(内层初始化变量; 内层判断条件; 内层变量更新){
​ 内层循环体;
​ }
}
外层循环执行一次，内层循环执行一轮。

对象

用来储存无长度、无顺序的数据，一般用来做前后台交互。
{键: 值, 键: 值} 键值对
键建议加引号，不加也能用。

for-in

for(var 变量 in 数据){
​ 变量: 键
​ 数据[变量]: 值
}
