css Hack

发表于 2021-08-09

此东西不常用，但是需要了解。

CSS Hack

  • 条件Hack

    • 大于：gt

    • 大于或等于：gte

    • 小于：lt

    • 小于或等于：lte

      ​ <!–[if IE]>

      ​ <![endif]–>

只有IE6以上，才能看到应用了test类的标签是红色文本

IE10以上已经将条件注释移除，使用时要注意

  • 属性级Hack

      • _ 下划线：选择IE6及以下
      • *：选择IE7及以下
      • \0：选择ie8以上

    • color:red;//所有浏览器可识别

    • _color:red;//仅IE6识别

    • *color:red;//IE6、IE7识别

    • color:red\0;//IE8、IE9识别

  • 选择符级Hack

    • *html .box { background:red; }//只有在IE6显示红色
