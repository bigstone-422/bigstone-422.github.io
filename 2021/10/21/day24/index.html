BootStrap

介绍

Bootstrap 是最受欢迎的 HTML、CSS 和 JS 框架，用于开发响应式布局、移动设备优先的 WEB 项目。
入门: 下载: 完整的bootstrap

快速使用

容器

容器 全页面所有内容的盒子
.container 在页面中间有1200px
.container-fluid 占满全页面
自带左右15px的内边距

响应式图片

栅格系统

使用

  1. 写行 row
  2. 列: col--

第一个: 屏幕尺寸
​ lg 超大屏幕: > 1200
​ md 中等屏幕: 992 < x < 1200
​ sm 小屏幕: 768 < x < 992
​ xs 超小屏幕: < 768
第二个: 占得比例 1-12
在当前尺寸以上 使用堆叠的布局方式 在当前尺寸以下 水平排布

兼容

兼容不同尺寸的栅格布局:
可以给一个元素添加多个栅格系统类名 在显示的时候就会根据不同尺寸的样式进行显示

offset

当前列需要在原来的位置上 向后推移几个位置 不会换行
col-* -offset-*

第一个: 屏幕尺寸
​ lg 超大屏幕: > 1200
​ md 中等屏幕: 992 < x < 1200
sm 小屏幕: 768 < x < 992
xs 超小屏幕: < 768
第二个: 占得比例 1-12

pull/push

push: 向后推几个格子
pull: 向前前进几个格子
col-* -push/pull-*

列嵌套

每一个列中可以再包一个行 将当前列作为行 继续划分12格
