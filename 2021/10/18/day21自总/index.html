JS学习第二十一天

发表于 2021-10-18

第二十一天学习了jQuery的元素宽高，操作节点，事件绑定等等。

day21自总

元素宽高

client: 可视宽高 innerWidth/innerHeight: 内容+内边距
offset: 占位宽高 outerWidth/outerHeight(布尔值) false 不传: 内容 + 内边距 + 边框 true: 内容 + 内边距 + 边框 + 外边距
scroll: 滚动距离

元素的位置

position: 获取到的结果是当前元素的定位距离 以对象方式返回
{
​ left: 0,
​ top: 0
}
offset: 获取的元素距离body的一个偏移距离

添加节点

  1. 创建节点: $(‘标签节点’);

  2. 父元素末尾:
    父.append(子);
    子.appendTo(‘选择器’);

  3. 父元素开头:
    父.prepend(子);
    子.prependTo(‘选择器’);

  4. 插入某个元素之前:
    参考节点.before(新节点);
    新节点.insertBefore(参考节点);

  5. 插入到某个元素之后:
    参考节点.after(新节点);
    新节点.insertAfter(参考节点);

删除节点

remove: 删除当前元素 返回被删除掉的元素 不包含原来元素的行为
detach: 删除当前元素 返回被删除掉的元素 包含原来元素的行为
empty: 清空元素 删除当前元素中所有子节点

克隆节点

clone(布尔): false 不传: 不克隆行为 不保留事件函数

替换节点

参考节点.replaceWith(新节点);
新节点.replaceAll(参考节点);

事件对象

on

  1. 给一个元素绑定一个事件
    jq对象.on(事件类型, 事件处理函数);

  2. 给一个元素的多个事件添加同一处理函数
    jq对象.on(‘事件 事件 事件…’, 事件处理函数)

  3. 给不同事件添加不同处理函数
    jq对象.on({
    ​ 事件: 函数,
    ​ 事件: 函数
    });

  4. 绑定自定义事件:
    由程序员自己定义的事件

  5. 事件委托
    jq对象(父).on(事件类型, ‘子选择器’, 事件处理函数);
    this指向的就是每一个子元素

one

one: 与on是一样的 只会触发一次

off

合成事件

jq对象.hover()
​一个函数: 滑入滑出都触发
​两个函数: 第一个滑入 第二个滑出

阻止冒泡和取消默认行为
